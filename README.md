## 图库使用方法

### 初始化本地仓库git init
- git add README.md
- git config --global user.email "you@example.com" （可以去掉--global）
- git config --global user.name "Your Name" （可以去掉--global）
- git commit -m "提交描述"

### 图片上传
- git remote add origin https://gitee.com/future-hu/images.git
- git push -u origin "master"
